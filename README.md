The Tic-Tac-Toe game is a classic board game, developed by two students from the Federal University of Uberl√¢ndia as a way to solidify their programming knowledge in the C language acquired in the classroom.

This project utilizes concepts from the C language, such as:

- Conditionals
- Loops
- Functions
- Arrays
- Strings
- Structs
- Pointers
- Dynamic Memory Allocation
- File Handling
- among others.

**How to Play**:
The objective of the game is to fill a 3x3 grid with alternating X and O symbols until one of the players manages to form a straight line horizontally, vertically, or diagonally.
